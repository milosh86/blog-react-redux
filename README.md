# blog-react-redux
## How to run
* npm install
* npm start
* navigate to http://localhost:3000

##What's in?
* React components (smart/containers and dumb/stateless)
* Module bundling and Babel transpilation with Webpack
* Hot reloading
* Redux state container
* React router
